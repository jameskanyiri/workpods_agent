# Status Review Workflows

## Project health review

1. Read project detail.
2. Identify blockers, overdue tasks, and missing structure.
3. Group active tasks by milestone.
4. Identify the highest-risk milestone or missing milestone.
5. Summarize what is done, what is missing, and what should happen next.

## "What's next?"

1. Read current project and tasks.
2. Find the nearest unblocked action.
3. Recommend the one next move with the highest leverage.
