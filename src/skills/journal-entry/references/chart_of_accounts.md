# Standard Kenyan SME Chart of Accounts

## Overview

This chart of accounts is designed for Kenyan SMEs following IFRS for SMEs. Account codes use a 4-digit numbering system with the first digit indicating the account class.

## Account Classes

| First Digit | Class | Normal Balance |
|-------------|-------|---------------|
| 1 | Assets | Debit |
| 2 | Liabilities | Credit |
| 3 | Equity | Credit |
| 4 | Revenue | Credit |
| 5 | Expenses | Debit |
