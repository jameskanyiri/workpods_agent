# Thread Workflows

## Choose the right communication object

1. If it is project-level progress for stakeholders, use `project-updates`.
2. If it is one task execution detail, use `comments`.
3. If it is an ongoing broader discussion, use `threads`.

## Attach a thread to a project

1. Read the thread and project context.
2. Confirm the thread is the right long-lived discussion object.
3. Attach it to the project.
